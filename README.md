# thetinywebagency.github.io
Website for Thetinywebagency

## GitHub Pages
https://thetinywebagency.github.io/

## Install
```
$ git clone https://github.com/thetinywebagency/thetinywebagency.github.io.git
$ cd thetinywebagency.github.io

$ npm install
```

## Build
```
# Production Build
$ npm run build

# Development Build
$ npm run build:dev
```
